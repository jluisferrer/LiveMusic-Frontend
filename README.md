### 1 - Local Installation:

- Clonar repositorio.

- ```bash
  git clone https://github.com/jluisferrer/LiveMusic-Frontend/
```
- Instala las dependencias del proyecto:

```bash
  npm install
```

- Lanza la APP:
```bash
npm run dev
```
